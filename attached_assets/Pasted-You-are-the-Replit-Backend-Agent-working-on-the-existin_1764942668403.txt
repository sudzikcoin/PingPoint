1. UPDATE TAILWIND THEME
────────────────────────

Open `tailwind.config.js` and extend the theme with custom colors and backgrounds.

In `theme.extend` add:

- Colors:
  - `brand-bg`: #050608  (app background, very dark)
  - `brand-card`: #101522 (main card background)
  - `brand-border`: #1f2633
  - `brand-text`: #e5e7eb
  - `brand-muted`: #9ca3af
  - `brand-gold`: #f5c550
  - `brand-gold-dark`: #c89628
  - `brand-gold-light`: #ffe9a9
  - `brand-dark-pill`: #1f2933
  - `brand-light-pill`: #f5f5f7

- Box shadows:
  - `shadow-pill-gold`: something like `0 10px 24px rgba(245,197,80,0.35)`
  - `shadow-pill-dark`: `0 10px 24px rgba(0,0,0,0.6)`

- Gradient backgrounds (via `backgroundImage`):
  - `pill-gold`: left slightly lighter → right slightly darker (brand-gold-light → brand-gold-dark)
  - `pill-dark`: dark grey / black gradient
  - `pill-light`: silver gradient

Example (adapt to your Tailwind config):

extend: {
  colors: {
    'brand-bg': '#050608',
    'brand-card': '#101522',
    'brand-border': '#1f2633',
    'brand-text': '#e5e7eb',
    'brand-muted': '#9ca3af',
    'brand-gold': '#f5c550',
    'brand-gold-dark': '#c89628',
    'brand-gold-light': '#ffe9a9',
    'brand-dark-pill': '#1f2933',
    'brand-light-pill': '#f5f5f7',
  },
  backgroundImage: {
    'pill-gold': 'linear-gradient(135deg, #ffe9a9, #f5c550, #c89628)',
    'pill-dark': 'linear-gradient(135deg, #111827, #111827, #020617)',
    'pill-light': 'linear-gradient(135deg, #ffffff, #f5f5f7, #e5e7eb)',
  },
  boxShadow: {
    'pill-gold': '0 10px 24px rgba(245,197,80,0.35)',
    'pill-dark': '0 10px 24px rgba(0,0,0,0.6)',
  },
}

Replace any old hard-coded blues/greys (e.g. `bg-blue-500`) with these new brand colors.

3. CREATE A REUSABLE PILL BUTTON COMPONENT
────────────────────────

Create `src/components/ui/PillButton.tsx`.

This must be a client component:

- At the top: `"use client";`

Props:

- `variant?: "gold" | "dark" | "light"` (default `"gold"`).
- `size?: "md" | "lg"` (optional).
- `icon?: React.ReactNode` (by default: the circular gold disc with arrow).
- `children?: React.ReactNode` (label text).
- `className?: string`
- Plus all standard `<button>` props (onClick, disabled, type, etc.).

Base button styles:

- `inline-flex items-center justify-between gap-3 rounded-full px-6 py-2 transition-all select-none`
- `cursor-pointer disabled:opacity-50 disabled:cursor-not-allowed`.

Variant styles:

1) GOLD:
   - `bg-pill-gold text-[#6b3b05] shadow-pill-gold`
   - maybe `font-semibold tracking-wide`.
   - hover: `hover:-translate-y-0.5 hover:shadow-[...]`.

2) DARK:
   - `bg-pill-dark text-brand-text shadow-pill-dark border border-black/40`.

3) LIGHT:
   - `bg-pill-light text-slate-700 shadow-md border border-white/40`.

Inside button, structure:

- Left: circular gold icon container:

  - `flex items-center justify-center w-8 h-8 rounded-full bg-gradient-to-br from-brand-gold-light via-brand-gold to-brand-gold-dark shadow-pill-gold`.

  Inside this div, render a simple right-arrow SVG (no external icon library).

- Right: label text:

  - `span` with `flex-1 text-center text-sm font-semibold tracking-wide`.

Rough JSX:

<button ...className={computedClasses}>
  <span className="flex items-center justify-center w-8 h-8 rounded-full ...">
    {/* inline SVG arrow */}
  </span>
  <span className="flex-1 text-center">
    {children}
  </span>
</button>

You may add support for `size="lg"` to increase padding and width for prominent CTAs.
